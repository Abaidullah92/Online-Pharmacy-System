# Online Pharmacy System (C++)

## 📌 Description
This is a console-based Online Pharmacy System developed in C++.  
It allows users to manage medicines, place orders, and handle inventory.

## 🚀 Features
- Add medicines
- Search medicines
- Place orders
- View inventory

## 🛠 Technologies Used
- C++
- File Handling

## ▶️ How to Run
1. Open project in any C++ compiler
2. Compile main.cpp
3. Run the program
